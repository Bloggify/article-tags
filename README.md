Article Tags
============

A Bloggify plugin that adds the functionality to add tags to articles.

## Configuration

```json
{
  "name": "article-tags",
  "source": "git@github.com:BloggifyPlugins/article-tags.git",
  "version": "2.0.0"
}
```
